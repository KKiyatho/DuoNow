# DuoNow 웹 테스트용 명령어

이 문서는 Flutter 웹 기준으로 DuoNow를 빠르게 실행하고 확인할 때 쓰는 명령어 모음입니다.

## 1) Flutter 설치 확인

```powershell
flutter --version
```

## 2) 의존성 설치

```powershell
flutter pub get
```

## 3) Firebase 설정 준비

Firebase 연결값은 코드에 이미 들어가 있으므로, 지금은 바로 웹 실행이 가능합니다.

선택적으로 다시 생성하고 싶을 때만 사용:

```powershell
dart pub global activate flutterfire_cli
flutterfire configure
```

직접 실행은 아래처럼 하면 됩니다.

```powershell
flutter run -d chrome
```

## 4) 웹 개발 서버 실행

Chrome에서 바로 띄우기:

```powershell
flutter run -d chrome
```

포트를 고정하고 싶을 때:

```powershell
flutter run -d chrome --web-port 5173
```

## 5) 웹 빌드 확인

```powershell
flutter build web
```

빌드 결과 미리보기:

```powershell
flutter pub global activate dhttpd
dhttpd --path build/web
```

## 6) 테스트 실행

```powershell
flutter test
```

정적 분석까지 같이 보고 싶을 때:

```powershell
flutter analyze
```

## 7) 자주 쓰는 빠른 루틴

1. `flutter pub get`
2. `flutter run -d chrome`
3. 로그인/회원가입/Google/전화번호 로그인 확인
4. 프로필 저장 확인
5. `flutter build web`
6. `flutter analyze`

## 8) 문제가 있을 때

캐시가 꼬였을 때:

```powershell
flutter clean
flutter pub get
```

웹에서 Firebase 연결 화면만 보일 때:

- `flutterfire configure`를 다시 실행
- `flutter run -d chrome --dart-define=...` 값 확인
- Realtime Database 규칙이 [database.rules.json](database.rules.json)인지 확인
